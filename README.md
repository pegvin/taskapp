## Building From Source

Get The Repository Using `git`:

```bash
git clone https://github.com/DEVLOPRR/taskapp
```

Change Directory To Cloned Folder And Install Dependencies:

```bash
cd taskapp
npm install
npm install -g @neutralinojs/neu
neu update
```

Build:

```bash
npm run build
```
